# Archland Linux

A set of scripts that configures Arch with Hyprland to make it a desktop Linux for daily use.

These scripts run on a fresh Arch Linux to complete the setup to be able to use it as a daily driver.

## Implementing features

To implement features check `omarchy` folder to see how they implemented and use it as a guide.
